## [0.1.0] - 2026-04-02

### Added

- time entry management: add, update, remove, and list commands
- harvest provider with full API integration
- auth commands: login, list, default, whoami, logout
- project and client discovery commands
- harvest api passthrough command for raw API access
- harvest schema command to dump OpenAPI spec
- json and robot output modes for scripting
